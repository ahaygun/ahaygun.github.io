02 / Projeler

Kişisel projeler.

CDR Fraud Detection

Canlı ↗ GitHub ↗

Akan CDR (çağrı kaydı) verisinden gerçek zamanlı telekom fraud tespiti — Kafka üzerinde event-driven bir pipeline, üç bağımsız fraud kuralını ayrı ayrı deploy edilebilir mikroservislerde çalıştırır.

  • Üç fraud kuralı, üç teknik: velocity (Redis kayan pencere), impossible-travel (stateful konum + gRPC), IRSF (premium harcama penceresi).
  • MSISDN-key'li partition + record_id idempotency ile at-least-once teslim pratikte effectively-once; enrichment düşse pipeline çalışmaya devam eder.
  • Helm ile Kubernetes'e deploy; KEDA tüketiciyi Kafka lag'ine göre 1→3 replica'ya ölçekliyor.
  • Prometheus + Grafana; ölçülmüş ~1.700 olay/s (enrichment cache ile ≈1.9×), p99 ~25 ms.

Fatura-Fiş Botu

Poster ↗

Bitirme projesi — Türkçe fiş ve faturaları OCR + NLP ile yapılandırılmış finansal veriye çeviren bir görsel işleme sistemi.

  • Go modüler monolit; ağır OCR/çıkarım işleri Redis + Asynq iş kuyruğuyla asenkron çalışır.
  • Görüntüden veriye 4 aşamalı pipeline: ön işleme → OCR → varlık çıkarımı → harcama sınıflandırma (OpenCV · Tesseract · BERTurk · ML).
  • Çok kiracılı PostgreSQL (tenant izolasyonu + audit) + MinIO depolama; güven skoruna göre otomatik onay ya da insan denetimi.
  • Go

  • PostgreSQL

Meeting to Mail

GitHub ↗

Go ile toplantı otomasyonu — sesi kaydeder, metne döker, LLM ile özetler ve katılımcılara mailler.

  • Çekirdeğinde PostgreSQL tabanlı, eşzamanlılığa güvenli bir iş kuyruğu (FOR UPDATE SKIP LOCKED).
  • Aşamalı pipeline (transcribe → summarize → send), otomatik migration ve SSE ile canlı ilerleme.
  • whisper.cpp + Ollama ile tamamen yerel de çalışabilir; API anahtarları opsiyonel.

API Gateway · Rate Limiter

GitHub ↗

Go ile bir API Gateway — backend servislerin önünde yönlendirme, auth ve rate limiting.
