02 / Projeler

Kişisel projeler.

Fatura-Fiş Botu

Poster ↗

Bitirme projesi — Türkçe fiş ve faturaları OCR ve NLP ile yapılandırılmış finansal veriye çeviren, çok kiracılı bir Akıllı Doküman İşleme (IDP) sistemi.

  • Go modüler monolit; Redis + Asynq iş kuyruğu ve worker havuzuyla ağır OCR/çıkarım işleri asenkron çalışır.
  • Uçtan uca pipeline: OpenCV ön işleme → Tesseract-LSTM OCR → BERTurk NER → SVM/RF ile harcama sınıflandırma.
  • PostgreSQL (tenant izolasyonu + audit) ve MinIO depolama; durum makinesi ve güven skoruna dayalı otomatik onay / insan denetimi.
  • Go
  • Vue
  • PostgreSQL
  • Redis
  • MinIO
  • Tesseract-LSTM
  • BERTurk

Meeting to Mail

GitHub ↗

Go ile toplantı otomasyonu — sesi kaydeder, metne döker, LLM ile özetler ve katılımcılara mailler.

  • Çekirdeğinde PostgreSQL tabanlı, eşzamanlılığa güvenli bir iş kuyruğu (FOR UPDATE SKIP LOCKED).
  • Aşamalı pipeline (transcribe → summarize → send), otomatik migration ve SSE ile canlı ilerleme.
  • whisper.cpp + Ollama ile tamamen yerel de çalışabilir; API anahtarları opsiyonel.
  • Go
  • Vue
  • PostgreSQL
  • whisper.cpp
  • Ollama

API Gateway · Rate Limiter

GitHub ↗

Go ile production-shaped bir API Gateway — backend servislerin önünde yönlendirme, auth ve rate limiting uygular.

  • İstemci bazlı token bucket rate limiting: in-memory ve Redis + Lua (dağıtık, atomik) iki backend; aşımda 429 + X-RateLimit-* header'ları.
  • Dayanıklılık: upstream başına circuit breaker + exponential backoff'lu retry.
  • API-key auth, Prometheus metrikleri, request-ID'li slog, graceful shutdown.
  • Tek komutla ayağa kalkan docker-compose demo (gateway + Redis + upstream).
  • Go
  • Redis
  • Lua
  • Prometheus
  • Docker
