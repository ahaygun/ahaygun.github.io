# ahaygun.github.io

Personal CV / portfolio site for **Ahmet Hasan Aygün** — a single-file, dependency-free page.

🔗 **Live:** https://ahaygun.github.io

## About

- **Single `index.html`** — all CSS and JS inline; no build step, no dependencies, no framework.
- **Bilingual (Türkçe / English)** — client-side toggle, preference stored in the browser (~lightweight i18n via `data-tr` / `data-en`).
- **Dark theme, responsive, mobile-first** — subtle scroll-reveal animations and micro-interactions.
- **Sections** — About · Experience · Projects · Skills · Education · Contact.
- **Deployed via GitHub Pages** — push to `main` to publish.

## Develop

Just open `index.html` in a browser, or serve it locally:

```bash
python3 -m http.server 8000   # then open http://localhost:8000
```
